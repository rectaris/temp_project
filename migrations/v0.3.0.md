# v0.3.0 Migration Notes

This release expands the generated workflow template and makes Copier validation reproducible through `uv`.

## Downstream Actions

1. Ensure the generated repository is clean with `git status --short`.
2. Run `copier update --vcs-ref v0.3.0`.
3. Review `.copier-answers.yml` changes.
4. Review and resolve any `*.rej` files.
5. Run the generated repository validation commands.
6. Commit the update as one coherent workflow-template refresh.

## Notes

- Plan manifest parsing and plan index updates now share `scripts/planlib.py`.
- Generated plan lifecycle scripts still expose the same shell entrypoints.
- External-service policy docs are generated consistently for enabled and disabled integrations.
- Template development now uses `uv` and the locked Copier version from `uv.lock`.
